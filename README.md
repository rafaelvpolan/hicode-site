# hicode-site

Site institucional do **hicode** — gerenciador de projetos autônomo open source.

Stack: **Vite + Vue 3 + TypeScript**. Landing page responsiva com SEO (meta/OG tags),
contador de estrelas ao vivo do GitHub e área de apoio/doação.

## Rodar local

```bash
npm install
npm run dev      # http://localhost:5173
```

## Build

```bash
npm run build    # type-check (vue-tsc) + bundle em dist/
npm run preview  # serve o build
```

Repositório do projeto: https://github.com/rafaelvpolan/hicode
